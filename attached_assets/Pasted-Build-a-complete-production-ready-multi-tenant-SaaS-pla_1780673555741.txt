Build a complete production-ready multi-tenant SaaS platform for home improvement businesses.

This is not a single website.

This is a reusable SaaS operating system that LaunchFlow can sell to multiple home improvement companies.

The first tenant must be AMO Rendering, but the application must NOT hardcode AMO Rendering anywhere in the core system.

AMO Rendering should only exist as seed tenant data.

The platform must support rendering companies, roofing companies, driveway companies, landscaping companies, window companies, plasterers, builders, and other local service businesses.

The system must include:

1. Public-facing website per tenant
2. Tenant admin dashboard
3. Customer portal
4. LaunchFlow SaaS owner dashboard
5. Lead management CRM
6. Quote management
7. Project management
8. Case study engine
9. Gallery engine
10. Reviews engine
11. Local SEO area page engine
12. Service page engine
13. Blog system
14. Visualiser request system
15. File upload system
16. Role-based access control

Do not build a prototype.

Do not leave placeholders.

Do not create dead buttons.

Do not create fake forms.

Do not create coming soon pages.

Do not use lorem ipsum.

Every form must submit.

Every dashboard action must update real database records.

Every tenant must have isolated data.

Every public website must be generated dynamically from tenant settings and tenant content.
